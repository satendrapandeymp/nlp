<para> Colors are very important to give a good look and feel to your website. You can specify colors on page level using <body> tag or you can set colors for individual tags using bgcolor attribute. </para>
<para> The <body> tag has following attributes which can be used to set different colors − </para>
<para> bgcolor − sets a color for the background of the page. </para>
<para> text − sets a color for the body text. </para>
<para> alink − sets a color for active links or selected links. </para>
<para> link − sets a color for linked text. </para>
<para> vlink − sets a color for visited links − that is, for linked text that you have already clicked on. </para>
<header> HTML Color Coding Methods </header>
<para> There are following three different methods to set colors in your web page − </para>
<para> Color names − You can specify color names directly like green, blue or red. </para>
<para> Hex codes − A six-digit code representing the amount of red, green, and blue that makes up the color. </para>
<para> Color decimal or percentage values − This value is specified using the rgb( ) property. </para>
<para> Now we will see these coloring schemes one by one. </para>
