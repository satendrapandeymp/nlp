<para> An HTML marquee is a scrolling piece of text displayed either horizontally across or vertically down your webpage depending on the settings. This is created by using HTML <marquees> tag. </para>
<para> Note − The <marquee> tag deprecated in HTML5. Do not use this element, instead you can use JavaScript and CSS to create such effects. </para>

<header> The <marquee> Tag Attributes </header>
<para> Following is the list of important attributes which can be used with <marquee> tag. </para>
<para> width </para>
<para> This specifies the width of the marquee. This can be a value like 10 or 20% etc. </para>
<para> height </para>
<para> This specifies the height of the marquee. This can be a value like 10 or 20% etc. </para>
<para> direction </para>
<para> This specifies the direction in which marquee should scroll. This can be a value like up, down, left or right. </para>
<para> behavior </para>
<para> This specifies the type of scrolling of the marquee. This can have a value like scroll, slide and alternate. </para>
<para> scrolldelay </para>
<para> This specifies how long to delay between each jump. This will have a value like 10 etc. </para>
<para> scrollamount </para>
<para> This specifies the speed of marquee text. This can have a value like 10 etc. </para>
<para> loop </para>
<para> This specifies how many times to loop. The default value is INFINITE, which means that the marquee loops endlessly. </para>
<para> bgcolor </para>
<para> This specifies background color in terms of color name or color hex value. </para>
<para> hspace </para>
<para> This specifies horizontal space around the marquee. This can be a value like 10 or 20% etc. </para>
<para> vspace </para>
<para> This specifies vertical space around the marquee. This can be a value like 10 or 20% etc. </para>
