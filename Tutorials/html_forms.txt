<para> HTML Forms are required, when you want to collect some data from the site visitor. For example, during user registration you would like to collect information such as name, email address, credit card, etc. </para>
<para> A form will take input from the site visitor and then will post it to a back-end application such as CGI, ASP Script or PHP script etc. The back-end application will perform required processing on the passed data based on defined business logic inside the application. </para>
<para> There are various form elements available like text fields, textarea fields, drop-down menus, radio buttons, checkboxes, etc. </para>
<para> The HTML <form> tag is used to create an HTML form and it has following syntax − </para>
<header> Form Attributes </header>
<para> Apart from common attributes, following is a list of the most frequently used form attributes − </para>
<para> action </para>
<para> Backend script ready to process your passed data. </para>
<para> method </para>
<para> Method to be used to upload data. The most frequently used are GET and POST methods. </para>
<para> target </para>
<para> Specify the target window or frame where the result of the script will be displayed. It takes values like _blank, _self, _parent etc. </para>
<para> enctype </para>
<para> You can use the enctype attribute to specify how the browser encodes the data before it sends it to the server. Possible values are − </para>
<para> application/x-www-form-urlencoded − This is the standard method most forms use in simple scenarios. </para>
<para> mutlipart/form-data − This is used when you want to upload binary data in the form of files like image, word file etc. </para>
<para> Note − You can refer to Perl & CGI for a detail on how form data upload works. </para>

<para> Password input controls − This is also a single-line text input but it masks the character as soon as a user enters it. They are also created using HTMl <input> tag. </para>
<para> Multi-line text input controls − This is used when the user is required to give details that may be longer than a single sentence. Multi-line input controls are created using HTML <textarea> tag. </para>
